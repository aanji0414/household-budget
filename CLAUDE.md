# CLAUDE.md

이 파일은 이 저장소에서 작업할 때 Claude Code(claude.ai/code)에게 제공되는 안내 문서입니다.

## 프로젝트 개요

점수 분석을 위한 Python 유틸리티 프로젝트입니다. 현재 `sample.py`에 기본 숫자 유틸리티 함수(`calculate_average`, `find_max`)가 포함되어 있으며, 샘플 점수 목록을 대상으로 동작을 시연합니다.

## 실행 방법

```bash
python sample.py
```

## 코드 컨벤션

- 함수는 숫자 리스트를 입력받아 단일 값을 반환합니다.
- 출력 메시지는 한글을 사용합니다 (예: `"평균:"`, `"최고점:"`).
