# CLAUDE.md

이 파일은 이 저장소에서 작업할 때 Claude Code(claude.ai/code)에게 제공되는 안내 문서입니다.

## 프로젝트 개요

점수 분석을 위한 Python 유틸리티 프로젝트입니다. 현재 `sample.py`에 기본 숫자 유틸리티 함수(`calculate_average`, `find_max`)가 포함되어 있으며, 샘플 점수 목록을 대상으로 동작을 시연합니다.

## 실행 방법

```bash
python sample.py
```

## 코드 컨벤션

- 함수는 숫자 리스트를 입력받아 단일 값을 반환합니다.
- 출력 메시지는 한글을 사용합니다 (예: `"평균:"`, `"최고점:"`).
- 워밍업이나 인사 문구로 답을 열지 마라. 매번 본론부터 시작해라.
- 행동 전에 2~3가지 접근을 먼저 보여줘라.
- 모르면 모른다고 해라.
- 답변 길이는 질문 크기에 맞춰라.
- 큰 변경 전에는 멈추고, 무엇을 왜 바꿀 건지 먼저 설명하기
- 묻지 않은 부분은 손대지 않기. 더 좋아 보여도 마지막에 코멘트만 남기고 건드리지 말 것
- 작업 끝나면 뭘 바꿨고 뭘 안 건드렸는지 한 줄 요약
- 내 이름으로 무언가를 보내고/올리고/예약하는 행동은, 그 메시지 안에서 명시적으로 "그래"라고 답하기 전엔 절대 실행하지 말 것
- 요청한 것만 건드린다. 버그 하나 고쳐달라고 했는데 변수명을 바꾸고 import를 재정리하고 옆 코드도 "개선"하는 일이 없어야 합니다.
- 파괴적 작업 전에는 무조건 확인. 파일 삭제, 코드 덮어쓰기, DB 레코드 drop 같은 건 되돌리기 어려워요. "무엇이 영향받는지" 먼저 나열하고, 그 메시지 안에서 명시적으로 동의 받기 전엔 진행하지 않기.
- 하드 스톱: 배포, DB 마이그레이션, 외부 API 호출, 이메일 전송 같은 건 절대 자동으로 실행 금지. 매번 동의를 받습니다.
